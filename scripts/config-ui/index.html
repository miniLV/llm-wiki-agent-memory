Wiki 结构和流程

minimal loop
  • 01 Source Logs Codex / Claude session logs原始会话日志是完整事实源；daily loop 生成一份 bounded Evidence Snapshot，并把同一份内容一次性交给 agent。
  • 02 Daily Wiki wiki/sources/ai-chats给人和 agent 看的每日中文 wiki；每个关键会话精确链接到 Snapshot Evidence Card。
  • 03 Reviewed Concepts wiki/conceptsDaily 只做 backup；weekly 二次 review 后，只有真正有价值的经验才进入 concepts，不生成 behavior rules。
  • 04 Apply Skill .agent/skills/engineering-memory-loader业务 repo 按需加载查询 skill，找历史背景和防踩坑规则。
