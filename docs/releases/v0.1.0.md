# LLM Wiki Agent Memory v0.1.0

首个可公开使用的版本：把 Codex / Claude Code 本机会话整理成可查询、可审计的 Obsidian 风格 Markdown Wiki。

## Highlights

- Local-first：原始 session 和生成的 Wiki 都保留在本机。
- Plain Markdown：无需向量数据库或托管服务。
- Capture → Daily → Reconcile → Retrieve：分离原始证据、每日整理、经验晋升和只读查询。
- Codex + Claude Code：支持两种本地 session source。
- Local setup UI：通过浏览器完成依赖检查、数据源配置和 Automations 引导。
- Auditable memory：所有长期记忆都可以直接阅读、编辑和版本管理。

## Quick start

```bash
git clone https://github.com/miniLV/llm-wiki-agent-memory.git
cd llm-wiki-agent-memory
bash scripts/config-ui.sh --open
```

## Notes

- macOS 是当前体验最完整的平台。
- Codex App Automations 是当前推荐的定时执行方式。
- 生成的 Daily Wiki 可能包含私人项目内容，公开仓库前请检查 `wiki/`。
